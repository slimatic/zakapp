# ZakApp Frontend

Your Secure, Private Wealth & Zakat Vault - Frontend Application

## Quick Start

```bash
docker run -p 3000:80 slimatic/zakapp-frontend:latest
```

## Features

- **Privacy First**: All calculations performed locally
- **Zero-Knowledge Architecture**: Your financial data never leaves your device
- **Multi-Madhab Compliance**: Toggle between Hanafi, Shafi'i, Maliki, and Hanbali rulings
- **Offline-First & PWA**: Works without internet connection
- **Responsive Design**: Optimized for all device sizes

## Environment Variables

- `REACT_APP_API_BASE_URL`: Backend API URL (default: /api)
- `REACT_APP_COUCHDB_URL`: CouchDB sync URL
