# Accessibility Features

> **Compliance Level**: WCAG 2.1 Level AA

## Key Features
- **Keyboard Navigation**: Full support for all interactive elements.
- **Screen Reader Support**: Tested with NVDA, JAWS, and VoiceOver.
- **Color Contrast**: Meets or exceeds 4.5:1 for normal text and 3:1 for large text.
- **Semantic HTML**: Proper use of landmarks, headings, and ARIA attributes.

## Audit Results
- **Lighthouse Accessibility Score**: 100/100
- **Manual Testing**: Verified across major browsers and assistive technologies.
