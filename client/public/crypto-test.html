Problem: Web Crypto API is blocked because you're accessing via IP address over HTTP.

Why: Browsers block crypto.subtle on insecure (HTTP) non-localhost origins for security.

Fix: Access ZakApp using one of these methods:
  • http://localhost:3005 (if running locally)
  • http://127.0.0.1:3005 (alternative localhost)
  • Set up HTTPS with a valid certificate
  • Use a reverse proxy with HTTPS (nginx, Traefik, Caddy)
