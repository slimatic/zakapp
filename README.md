# zakapp-spec

This repository contains the specification for zakapp, a user-friendly and visually appealing Zakat application designed for personal asset management and Zakat calculation.

## Short Description

zakapp is a user-friendly, self-hosted Zakat application with a modern UI. Features include secure login, flexible asset snapshots (lunar/solar), questionnaire-based asset determination, net worth/Zakat calculation, year-to-year accounting, and encrypted JSON data storage. The project is inspired by SimpleZakatGuide.com and will be developed using a spec-driven approach and deployed with Docker.
